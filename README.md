# Two-Gether 電商平台 

## Demo

https://caleb-liao.github.io/two-gether_e-commerce_website/dist

## 簡介

- 此為由「脫離邊緣人」為發想所設計的電子商務網站實作專案，目的為希望大家都能找到一起出遊玩的朋友。
- 包含前台及登入至後台的功能。
- RWD設計
- SCSS及api分檔管理
- 內容文案皆為虛構。

## 技術

- Vue.js
- Vue CLI
- Vue Router
- SCSS
- Element-UI
- RWD
